Gegeben ist ein String s und eine natürliche Zahl n. Sie wurden beauftragt, eine Funktion zu implementieren, die eine Liste aller Wörter aus dem String s zurückgibt, die genau n Konsonanten enthalten, in der Reihenfolge, in der diese Wörter im String s erscheinen. Wenn der String s leer ist, sollte die Funktion eine leere Liste zurückgeben. Hinweis: Sie können davon ausgehen, dass der Eingangsstring nur Buchstaben und Leerzeichen enthält.
